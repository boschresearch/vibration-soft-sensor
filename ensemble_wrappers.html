class softsensor.ensemble_wrappers.AsyncEnsemble(model, ensemble_weights)[source]ÔÉÅ

Async ensemble wrapper for point prediction models. Ensemble members predict the entire time series independently.

Parameters:

  • model (uncertainty model) ‚Äì model that is used for prediction

  • ensemble_weights (list[torch state_dict]) ‚Äì weights of the individual base models

Return type:

None.

Examples

>>> import softsensor.ensemble_wrappers as ew
>>> import softsensor.autoreg_models as am
>>> import torch
>>> m = []
>>> for i in range(5):
>>>     m_temp = am.ARNN(2, 1, 10, 10, [16, 8])
>>>     m.append(m_temp.state_dict())
>>> ensemble = ew.AsyncEnsemble(am.ARNN(2, 1, 10, 10, [16, 8]), m)
>>> input = torch.randn(32, 2, 10)
>>> rec_input = torch.randn(32, 1, 10)
>>> out = ensemble.estimate_uncertainty_mean_std(input, rec_input)
>>> print(out[0]) # Mean
>>> print(out[1]) # Std
prediction(dataloader, device='cpu', sens_params=None)[source]ÔÉÅ

Prediction of a whole Time Series

Parameters:

  • dataloader (Dataloader) ‚Äì Dataloader to predict output

  • device (str, optional) ‚Äì device to compute on. The default is ‚Äòcpu‚Äô.

  • sens_params (dict, optional) ‚Äì Dictionary that contains the parameters for the sensitivity analysis. Key ‚Äòmethod‚Äô defines the method for sensitivity analysis: ‚Äògradient‚Äô or ‚Äòperturbation‚Äô. Key ‚Äòcomp‚Äô defines whether gradients are computed for sensitivity analysis. Key ‚Äòplot‚Äô defines whether the results of the sensitivity analysis are visualized. Key ‚Äòsens_length‚Äô defines the number of randomly sampled subset of timesteps for the analysis. (If not a multiple of the model‚Äôs forecast, the number will be rounded up to the next multiple.) The default is None, i.e. no sensitivity analysis is computed.

Returns:

  • if comp_sens is False

  • torch.Tensor (Tensor of same langth as input, containing the predictions.)

  • if comp_sens is True ‚Äì (torch.Tensor, dict) : Tuple of Tensor of same length as input and sensitivity dict. Key is the prediction type, value is the sensitivity tensor.

Examples

Based on the Example from initialisation

>>> import softsensor.meas_handling as ms
>>> import numpy as np
>>> import pandas as pd
>>> t = np.linspace(0, 1.0, 101)
>>> d = {'inp1': np.random.randn(101),
         'inp2': np.random.randn(101),
         'out': np.random.randn(101)}
>>> handler = ms.Meas_handling([pd.DataFrame(d, index=t)], ['train'],
                               ['inp1', 'inp2'], ['out'], fs=100)
>>> loader = handler.give_list(window_size=10, keyword='training',
                               rnn_window=10, batch_size=1)

class softsensor.recurrent_models.AR_RNN(input_channels, pred_size, window_size, rnn_window, blocks, num_layers, blocktype='LSTM', hidden_size=None, activation='relu', bias=True, dropout=None, forecast=1, Pred_Type='Point')[source]ÔÉÅ

Autoregressive Recurrent Network that utilises the past outputs in combination with hidden cells

Parameters:

  • input_channels (int) ‚Äì Number of input channels

  • pred_size (int) ‚Äì Number of predicted values

  • window_size (int) ‚Äì Size of the sliding window applied to the time series

  • rnn_window (int) ‚Äì Window Size of the Recurent Connection

  • blocks (int) ‚Äì Number of parallel recurrent blocks.

  • num_layers (int) ‚Äì nunmber of recurernt blocks.(depth of the recurretn part)

  • blocktype (str, optional) ‚Äì blocktype, options are: ‚ÄòRNN‚Äô, ‚ÄòGRU‚Äô and ‚ÄòLSTM‚Äô. The default is ‚ÄòLSTM‚Äô.

  • hidden_size (list of int or None, optional) ‚Äì List gives the size of hidden units. The default is None.

  • activation (str, optional) ‚Äì Activation function to activate the feature space. The default is ‚Äòrelu‚Äô.

  • bias (bool, optional) ‚Äì If True, bias weights are used. The default is True.

  • dropout (float [0,1], optional) ‚Äì Adds dropout layers after each Linear Layer. The default is None.

Return type:

None.

Examples

>>> import softsensor.recurrent_models
>>> import torch
>>> m = softsensor.recurrent_models.AR_RNN(2, 1, 10, 10, 16, 1)
>>> print(m)
AR_RNN(
  (RecBlock): _LSTM(
    (lstm): LSTM(30, 16, batch_first=True)
  )
  (DNN): Feed_ForwardNN(
    (DNN): Sequential(
      (0): Linear(in_features=16, out_features=1, bias=True)
    )
  )
)
>>> input = torch.randn(32, 2, 10)
>>> rec_input = torch.randn(32, 1, 10)
>>> output = m(input, rec_input)
>>> print(output.shape)
torch.Size([32, 1, 1])

Prediction of a whole Time Series

Parameters:

  • dataloader (Dataloader) ‚Äì Dataloader to predict output

  • device (str, optional) ‚Äì device to compute on. The default is ‚Äòcpu‚Äô.

  • sens_params (dict, optional) ‚Äì Dictionary that contains the parameters for the sensitivity analysis. Key ‚Äòmethod‚Äô defines the method for sensitivity analysis: ‚Äògradient‚Äô or ‚Äòperturbation‚Äô. Key ‚Äòcomp‚Äô defines whether gradients are computed for sensitivity analysis. Key ‚Äòplot‚Äô defines whether the results of the sensitivity analysis are visualized. Key ‚Äòsens_length‚Äô defines the number of randomly sampled subset of timesteps for the analysis. (If not a multiple of the model‚Äôs forecast, the number will be rounded up to the next multiple.) The default is None, i.e. no sensitivity analysis is computed.

Returns:

  • if comp_sens is False

  • torch.Tensor (Tensor of same langth as input, containing the predictions.)

  • if comp_sens is True ‚Äì (torch.Tensor, dict) : Tuple of Tensor of same length as input and sensitivity dict. Key is the prediction type, value is the sensitivity tensor.

class softsensor.recurrent_models.RNN_DNN(input_channels, pred_size, window_size, blocks, num_layers, blocktype='LSTM', hidden_size=None, activation='relu', bias=True, dropout=None, forecast=1, Pred_Type='Point')[source]ÔÉÅ

Recurrent Network that utilises a hidden state

Parameters:

  • input_channels (int) ‚Äì Number of input channels

  • pred_size (int) ‚Äì Number of predicted values

  • window_size (int) ‚Äì Size of the sliding window applied to the time series

  • blocks (int) ‚Äì Number of parallel recurrent blocks.

  • num_layers (int) ‚Äì nunmber of recurernt blocks.(depth of the recurretn part)

  • blocktype (str, optional) ‚Äì blocktype, options are: ‚ÄòRNN‚Äô, ‚ÄòGRU‚Äô and ‚ÄòLSTM‚Äô. The default is ‚ÄòLSTM‚Äô.

  • hidden_size (list of int or None, optional) ‚Äì List gives the size of hidden units. The default is None.

  • activation (str, optional) ‚Äì Activation function to activate the feature space. The default is ‚Äòrelu‚Äô.

  • bias (bool, optional) ‚Äì If True, bias weights are used. The default is True.

  • dropout (float [0,1], optional) ‚Äì Adds dropout layers after each Linear Layer. The default is None.

Return type:

None.

Examples

>>> import softsensor.recurrent_models
>>> import torch
>>> m = softsensor.recurrent_models.RNN_DNN(2, 1, 10, 16, 1)
>>> print(m)
RNN_DNN(
  (RecBlock): _LSTM(
    (lstm): LSTM(20, 16, batch_first=True)
  )
  (DNN): Feed_ForwardNN(
    (DNN): Sequential(
      (0): Linear(in_features=16, out_features=1, bias=True)
    )
  )
)
>>> input = torch.randn(32, 2, 10)
>>> output = m(input)
>>> print(output.shape)
torch.Size([32, 1, 1])

>>> import softsensor.meas_handling as ms
>>> import numpy as np
>>> import pandas as pd
>>> t = np.linspace(0, 1.0, 101)
>>> d = {'inp1': np.random.randn(101),
         'inp2': np.random.randn(101),
         'out': np.random.randn(101)}
>>> handler = ms.Meas_handling([pd.DataFrame(d, index=t)], ['train'],
                               ['inp1', 'inp2'], ['out'], fs=100)
>>> loader = handler.give_list(window_size=10, keyword='training',
                               batch_size=32, Add_zeros=True)
>>> pred = m.prediction(loader[0])
>>> print(pred.shape)
torch.Size([1, 101])

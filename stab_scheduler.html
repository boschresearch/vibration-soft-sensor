Defined stability criterion for Autoregressive Neural Networks. If criterion < 0 Network is Input to state stable according to: https://www.researchgate.net/publication/346701375_Stability_of_discrete-time_feed-forward_neural_networks_in_NARX_configuration

Parameters:

Raises:

Warning ‚Äì if activation function is not defined with Lipschitz constant explicitly.

stabiliyt criterion for the Network.

class softsensor.stab_scheduler.log_lin_stab(arnn, s0=1e-06, s1=0.001, m=None, track=True, track_n=1)[source]ÔÉÅ

Stability score scheduler designed as an exponential function up to a stability criterion of zero and a linear function afterwards If criterion < 0 Network is Input to state stable according to: https://www.researchgate.net/publication/346701375_Stability_of_discrete-time_feed-forward_neural_networks_in_NARX_configuration

Parameters:

  • model (ARNN) ‚Äì ARNN according to softsensor.autoregr_models design

  • s0 (float, optional) ‚Äì stability parameter at the lower Networkspecific threshold, The default is 1e-6.

  • s1 (float, optional) ‚Äì stability parameter at stability score zero,The default is 1e-3.

  • m (float, optional) ‚Äì gradient of the linear function between [0, inf]. If None is given, use the gradient of the exponential function. The default is None.

  • track (Bool, optional) ‚Äì if True the values are tracked in a dict. The default is True.

Return type:

None.
